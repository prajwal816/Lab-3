## About The Project

![Nutriflow Screenshot](img/hero-min.png)

Nutriflow is a modern, responsive website for an AI-powered healthy meal delivery service. The platform offers personalized meal plans tailored to individual tastes and nutritional needs, making healthy eating convenient and accessible for everyone.

**Key Highlights:**

- AI-powered meal planning that adapts to your preferences and dietary requirements
- 365-day meal subscription service with flexible delivery options
- Support for various diets including vegetarian, vegan, keto, paleo, and more
- Beautiful, responsive design that works seamlessly across all devices
- Modern web technologies ensuring fast loading and smooth user experience

The website showcases a complete meal delivery service with sections for how it works, meal options, customer testimonials, pricing plans, and a contact form for new subscribers.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You only need a modern web browser and a local web server (optional for development).

For development with live reload:

- **Live Server Extension** (VS Code)
- **Python HTTP Server**
  ```sh
  python -m http.server 8000
  ```
- **Node.js HTTP Server**
  ```sh
  npm install -g http-server
  http-server
  ```

### Installation

1. Clone the repository

   ```sh
   git clone https://github.com/yourusername/nutriflow.git
   ```

2. Navigate to the project directory

   ```sh
   cd nutriflow
   ```

3. Open with a local server or directly in browser

   ```sh
   # Using Python
   python -m http.server 8000

   # Using Node.js
   npx http-server

   # Or simply open index.html in your browser
   ```

4. Visit `http://localhost:8000` in your browser
